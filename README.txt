はみがき90びょう v1-flat

iPhoneからGitHubへアップロードしやすいように、assetsフォルダなしの直置き版です。

GitHubにアップロードするファイル：
- index.html
- style.css
- app.js
- manifest.webmanifest
- hamigaki_song.m4a

README.txtはアップロードしてもしなくてもOKです。

使い方：
1. index.html を開く
2. iPhone/Android/PCで横向きにする
3. スタートを押す

内容：
- 添付音源を内蔵
- 90秒タイマー
- 15秒ごとの磨く場所切り替え
- 横画面向けUI
- キャラ未設定のため、歯ブラシと歯の仮アニメーション
